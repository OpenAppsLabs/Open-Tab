Open Tab is a high-performance, minimalist New Tab replacement designed for Chrome and Edge. It transforms the default browser landing page into a focused, privacy-respecting dashboard. No tracking. No bloat. Just a clean, professional space for your daily browsing with built-in essential utilities.

---

## 🚀 Key Features

- **Smart Search Engine Router:** Seamlessly switch between Google, DuckDuckGo, and Bing. Supports direct URL navigation.
- **Dynamic Quick Links:** Automatic high-res favicon fetching with drag-and-drop reordering.
- **Offline QR Utility:** Built-in "Send to Phone" tool. Generate QR codes instantly and 100% offline.
- **Adaptive UI/UX:** System-aware Light and Dark modes with smooth CSS transitions.
- **Fixed-Layout Architecture:** Prevents UI shifting during interaction for a premium feel.
- **Power User Optimized:** `1-7` number keys for shortcuts and `/` to focus search instantly.
- **Native Voice Search:** Integrated speech-to-text functionality.

## 📦 Installation

> [!WARNING]  
> **Important:** Modern browsers now block direct `.crx` file installs with a `CRX_REQUIRED_PROOF_MISSING` error. To bypass this, follow the manual installation steps below.

### Recommended: Manual Folder Installation

1. **Download** the source code (`.zip`) from the [Latest Release](https://github.com).
2. **Extract** the ZIP file into a dedicated folder on your computer.
3. Open your browser and navigate to `chrome://extensions/` (or `edge://extensions/`).
4. Enable **Developer mode** using the toggle in the top right corner.
5. Click the **Load unpacked** button.
6. Select the **folder** where you extracted the files.

### Alternative: CRX Method (May be blocked)
1. Download the `Open-Tab.crx` file.
2. Open `chrome://extensions/` and enable **Developer mode**.
3. **Drag and drop** the `.crx` file onto the page. 
4. *If you see a "Package is invalid" error, use the **Manual Folder Installation** method above.*
